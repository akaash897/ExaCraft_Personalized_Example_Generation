\begin{abstract}
Educational content personalization traditionally relies on static user profiles, failing to adapt to learners' evolving needs and real-time learning behaviors. We demonstrate an AI Example Generator system that addresses this limitation through dynamic learning context adaptation. Our system seamlessly integrates a Chrome browser extension, Python Flask API server, and Google Gemini AI to generate personalized examples that evolve with user interaction patterns. The key innovation lies in tracking five categories of dynamic learning context—including struggle indicators, mastery patterns, topic history, session boundaries, and learning progression signals—rather than relying solely on predetermined user characteristics. The demonstration showcases real-time adaptation scenarios where example complexity and content adjust based on behavioral signals such as topic repetition patterns and progression speed. This represents a significant advancement in educational AI applications, introducing novel approaches to behavioral learning analytics that enhance learning effectiveness through context-aware personalization.
\end{abstract}

\section{Introduction}

\subsection{Motivation}

Educational content personalization has become increasingly important as online learning platforms seek to improve learning outcomes through individualized instruction. However, current approaches predominantly rely on static user profiles—such as education level, profession, or stated preferences—that fail to capture the dynamic nature of learning processes. This creates significant limitations in real-world educational scenarios.

Learners' needs evolve continuously based on their comprehension, struggles with specific concepts, progression patterns, and contextual factors that change throughout learning sessions. A graduate-level software engineer may struggle with basic machine learning concepts while excelling in advanced programming topics, requiring different support strategies at different times. Similarly, the same learner may need simplified examples when encountering new domains but prefer advanced explanations for familiar concepts.

Existing educational AI systems typically treat personalization as a static matching problem, where predefined user categories are mapped to predetermined content types. This approach ignores crucial behavioral signals that indicate learning effectiveness, such as repeated requests for clarification, rapid topic progression indicating mastery, or interaction patterns that suggest cognitive load. The lack of real-time behavioral adaptation leads to suboptimal learning experiences where content complexity and presentation style remain fixed despite changing learner needs.

\subsection{Our System}

We present an AI Example Generator system that addresses these limitations through dynamic learning context adaptation. Our system operates as a Chrome browser extension that generates personalized examples for any highlighted text, seamlessly integrating with users' natural web browsing workflows. The system is supported by a Python Flask API that combines Google Gemini AI with sophisticated learning analytics to provide contextually-aware content generation.

The core innovation lies in tracking five categories of dynamic learning context that evolve based on real user interactions: recent topic history with timestamps, struggle indicators through repetition patterns and regeneration requests, mastery signals via progression speed, session-level boundary tracking with continuity, and behavioral signal integration across learning episodes. This behavioral learning analytics approach enables the system to adapt content complexity, explanation style, and example domains based on actual learning behaviors rather than static profile assumptions.

\subsection{Contributions}

This demonstration makes the following key contributions to educational AI research and practice:

\begin{itemize}
    \item \textbf{Dynamic Learning Context Framework}: A novel approach to educational personalization that tracks behavioral learning patterns across five contextual dimensions, moving beyond static user profile matching to real-time adaptation based on learning behaviors.
    
    \item \textbf{Seamless Workflow Integration}: A browser extension implementation that provides zero-disruption integration with natural web browsing behaviors, addressing a critical adoption barrier in educational technology tools.
    
    \item \textbf{Behavioral Learning Analytics}: Introduction of struggle detection through multi-signal analysis (topic repetition, regeneration patterns, interaction timing) and mastery identification through progression speed indicators, providing more reliable learning state assessment than single-metric approaches.
    
    \item \textbf{Cross-Session Learning Continuity}: A system design that maintains learning context across multiple browsing sessions, enabling long-term learning progressions that adapt to user development over time.
    
    \item \textbf{Live System Demonstration}: A fully deployed system that showcases practical applications of dynamic personalization in real educational scenarios, demonstrating the feasibility and effectiveness of behavioral learning analytics.
\end{itemize}

\section{Related Work}

Educational personalization has been extensively studied in adaptive learning systems and intelligent tutoring systems. Traditional approaches focus on learner models based on knowledge states and skill assessments \cite{bayly2024personalized}. However, these systems typically rely on explicit assessments rather than implicit behavioral signals during natural learning activities.

Recent work in AI-driven educational content generation has explored using large language models for personalized content creation \cite{kasneci2023chatgpt}. While these systems demonstrate the potential for AI-generated educational materials, they primarily use static user profiles without incorporating real-time learning behavior adaptation.

Learning analytics research has investigated behavioral pattern recognition in online educational environments \cite{yagci2022educational}. Zhang et al. \cite{saqr2023temporal} explored session-based learning pattern recognition, focusing on temporal aspects of learning behaviors. However, existing approaches typically analyze learning patterns post-hoc for course improvement rather than providing real-time content adaptation.

Educational technology tools have increasingly focused on seamless integration with learners' natural workflows. Browser extensions and web-based learning tools have gained popularity for their non-disruptive approach to educational support. However, most existing tools provide static functionality without behavioral adaptation capabilities.

Struggle detection in educational systems has been primarily based on performance metrics such as quiz scores and completion times \cite{baker2019educational}. Our work extends this by introducing multi-signal behavioral analysis that includes interaction patterns, regeneration requests, and topic revisitation frequencies as indicators of learning difficulties.

Our system distinguishes itself by combining real-time behavioral learning analytics with seamless workflow integration, providing dynamic content adaptation based on implicit learning signals rather than explicit assessments or static user characteristics.

\section{System Architecture and Implementation}

Figure 1 illustrates the system architecture showing the interaction flow between the browser extension, API server, and dynamic learning context engine (system diagram to be included in camera-ready version).

\subsection{Core Components}

The AI Example Generator consists of three integrated components working in concert to deliver personalized learning experiences:

\textbf{Browser Extension (Manifest V3):} A Chrome extension providing seamless integration with web browsing workflows. Users highlight any text on webpages and generate contextual examples through right-click context menus or keyboard shortcuts. The extension maintains local user profile management, learning session controls, and real-time behavioral signal detection through regeneration patterns, interaction timing, and topic progression tracking.

\textbf{Python Flask API Server:} A RESTful backend service that orchestrates AI content generation with behavioral learning analytics. The server manages persistent user profiles, tracks learning contexts across sessions, and integrates with Google Gemini AI through LangChain for sophisticated prompt engineering. Key endpoints support adaptive example generation, session management, learning signal recording, and cross-device synchronization.

\textbf{Dynamic Learning Context Engine:} A novel component that tracks and analyzes five categories of learning behavior in real-time: recent topic history with timestamped interactions, struggle indicators through topic repetition patterns and regeneration requests, mastery signals via quick progression through diverse topics, session-level boundary tracking with cross-session continuity, and behavioral signal integration for adaptive content generation. This engine continuously updates the learning model to inform content adaptation decisions.

\subsection{Technical Innovation}

The system's core innovation lies in its dynamic learning context adaptation mechanism. Unlike traditional educational AI systems that rely on static user profiles, our approach continuously analyzes user behavior to build contextual learning models.

\textbf{Behavioral Learning Analytics:} The system detects learning struggles primarily through topic repetition frequency (≥3 requests for the same topic) and explicit regeneration button clicks in the browser extension. Conversely, mastery is identified through quick progression across diverse topics (≥3 different topics in recent history) with minimal repetition.

\textbf{Session-Level Context Tracking:} Learning sessions are automatically managed with clear boundaries, allowing the system to understand learning continuity and provide context-aware adaptations. Each session maintains its own topic progression, struggle indicators, and temporal learning patterns.

\textbf{Multi-Dimensional Personalization:} The system incorporates user profiles across cultural, educational, professional, and preference dimensions while adapting these static elements based on real-time learning behavior.

\section{Live Demonstration Walkthrough}

The demonstration will showcase the AI Example Generator through three interactive scenarios that highlight key behavioral adaptation features. Attendees will observe the system in action through a live browser session with pre-configured user profiles and learning contexts.

\subsection{Scenario 1: Adaptive Complexity Progression}

\textbf{Demo Setup:} The demonstration will begin with a simulated user (Profile: Graduate Student, Computer Science) browsing a machine learning tutorial webpage. Attendees will observe the following interaction sequence:

\begin{enumerate}
    \item User highlights "machine learning" → System generates an intermediate-level example appropriate to their profile (Graduate Student, Computer Science)
    \item User quickly progresses to "neural networks" and "deep learning" → System detects quick progression pattern and adapts future responses with more advanced content
    \item User returns to "machine learning" and clicks regenerate button twice → System records struggle signals and the next generation uses simplified prompt engineering with foundational explanations
    \item Live display shows the learning context JSON file updating with topic history, struggle indicators, and mastery patterns
\end{enumerate}

\textbf{Key Observable Features:} Attendees will see how the system's responses evolve from technical to simplified based on behavioral signals rather than static profile settings.

\subsection{Scenario 2: Session-Aware Learning Continuity}

\textbf{Demo Setup:} The demonstration will showcase cross-session learning continuity through a multi-part scenario:

\begin{enumerate}
    \item User initiates a focused learning session on "data structures" 
    \item System generates examples and maintains topic progression history
    \item Session is paused and resumed after 5 minutes → System recalls previous context and builds upon earlier concepts
    \item User encounters difficulty with "binary trees" → System provides targeted support referencing previously successful "arrays" examples
\end{enumerate}

\textbf{Key Observable Features:} Live visualization of the session context engine will show how learning continuity is maintained and leveraged for adaptive support.

\subsection{Scenario 3: Multi-Signal Struggle Detection}

\textbf{Demo Setup:} The final scenario demonstrates sophisticated struggle detection through multiple behavioral signals:

\begin{enumerate}
    \item User explores "quantum computing" with normal interaction patterns → Standard complexity examples
    \item User exhibits struggle signals: long pauses (>30 seconds), multiple regenerations, topic revisitation → System automatically adjusts to provide simpler analogies and foundational concepts
    \item System explanation shows how multiple signals combine for reliable struggle detection
\end{enumerate}

\textbf{Key Observable Features:} Real-time demonstration of behavioral signal detection through console logs and JSON data visualization showing how struggle signals trigger simplified response generation.

Each scenario will include live screenshots of the browser extension interface, API response visualizations, and the dynamic learning context engine's internal state to provide complete transparency into the system's behavioral adaptation mechanisms.

\section{Implementation and Deployment}

The system is implemented using modern web technologies ensuring broad compatibility and ease of deployment. The Chrome extension uses Manifest V3 specifications for enhanced security and performance, while the Python Flask API leverages Google Gemini AI through LangChain for robust natural language processing capabilities.

All user data is stored locally in Chrome storage with optional server synchronization, ensuring privacy control while enabling cross-device learning continuity. The system includes comprehensive error handling and fallback mechanisms to ensure reliable operation even when AI services are temporarily unavailable.

The demonstration provides attendees with direct system interaction through the live browser extension, enabling firsthand experience of how dynamic learning context adaptation enhances traditional static personalization approaches. The system is actively deployed with documented learning contexts for multiple test users, demonstrating practical applicability in real learning scenarios.

\section{Conclusion}

We have demonstrated an AI Example Generator system that addresses the limitations of static educational personalization through dynamic learning context adaptation. The system's novel behavioral learning analytics approach—tracking struggle indicators, mastery patterns, and session continuity—enables real-time content adaptation that responds to actual learning behaviors rather than predetermined user categories.

The live demonstration showcases practical applications of this approach through seamless browser integration and multi-signal behavioral analysis. By combining Google Gemini AI with sophisticated learning analytics, the system provides a foundation for next-generation educational AI applications that adapt to learners' evolving needs.

Future work will focus on expanding cross-platform deployment, integrating collaborative learning patterns, and developing comprehensive evaluation frameworks for behavioral learning analytics in educational contexts.
